# Torch Ihmehimmeli
Torch Ihmehimmeli is a torch version of [ihmehimmeli](https://github.com/google/ihmehimmeli)
It is under development now.

Install cuda extension by

    sh install.sh

Run Example:

    python mnist_torch.py
